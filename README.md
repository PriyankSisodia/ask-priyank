# Ask Priyank

A personal AI digital twin of Priyank Sisodia — built using LLMs, Gradio, Google Sheets memory, and OpenAI-compatible APIs.

Live Demo: https://ask-priyank.onrender.com

---

## Features

* Conversational AI twin
* Human-like responses
* Google Sheets based memory/context
* Unknown question logging
* Contact/email capture
* OpenAI-compatible architecture
* Deployable on Render
* Works with Gemini, Groq, OpenRouter, etc.

---

## Tech Stack

* Python
* Gradio
* Google Sheets API
* Gemini / Groq APIs
* OpenAI-compatible SDK
